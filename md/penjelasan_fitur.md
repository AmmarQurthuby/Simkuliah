# Penjelasan Cara Kerja Fitur

## 1. Login
Pengguna memasukkan username dan password. Sistem memvalidasi ke database, lalu mengarahkan ke dashboard sesuai peran (admin, dosen, mahasiswa).

## 2. Dashboard Admin
- Terdapat sidebar navigasi ke data Mahasiswa, Dosen, Mata Kuliah, Jadwal, dan Nilai.
- Admin dapat menambah mahasiswa melalui form, dan melihat data mahasiswa dalam tabel.
- Fitur hapus mahasiswa tersedia dengan konfirmasi.

## 3. Dashboard Dosen
- Sidebar navigasi ke Input Nilai dan Jadwal.
- Dosen dapat memilih mata kuliah yang diampu, memasukkan NIM mahasiswa, nilai, dan semester.
- Data nilai yang sudah diinput ditampilkan dalam tabel.

## 4. Dashboard Mahasiswa
- Sidebar navigasi ke Nilai dan Jadwal.
- Mahasiswa dapat melihat daftar nilai yang sudah diinput dosen, lengkap dengan nama mata kuliah dan semester.

## 5. Logout
- User dapat keluar dari aplikasi, dan akan diarahkan ke halaman login dengan pesan sukses.

---
Setiap fitur didesain agar mudah digunakan, responsif, dan tampil modern dengan header, sidebar, card, dan footer.
