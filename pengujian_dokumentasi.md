# Tahap 5: Pengujian & Dokumentasi

## 1. Pengujian Lokal
- Jalankan aplikasi di browser: `http://localhost/si-perkuliahan/`
- Uji login sebagai Admin, Dosen, Mahasiswa.
- Uji fitur CRUD (tambah, edit, hapus data mahasiswa, dosen, mata kuliah, nilai).
- Uji input nilai oleh dosen dan cek tampilan nilai oleh mahasiswa.

## 2. Dokumentasi Screenshot
- Ambil screenshot halaman login, dashboard, form tambah data, tabel data, input nilai, dan tampilan nilai mahasiswa.

## 3. Catatan Kendala
- Catat masalah teknis yang ditemui (misal: error koneksi, validasi form, dsb) dan solusinya.

## 4. (Opsional) Deploy Online
- Jika diunggah ke hosting, catat langkah dan kendala upload.
